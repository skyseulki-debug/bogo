① 설정 방법 (무료 · 5분 소요)
1. console.firebase.google.com 접속 후 구글 로그인
2. 프로젝트 추가 → 이름 입력 → 프로젝트 만들기
3. 왼쪽 메뉴 빌드 → Realtime Database 클릭
4. 데이터베이스 만들기 → 테스트 모드 → 완료
5. 상단에 표시된 URL 복사
   예: https://프로젝트명-default-rtdb.firebaseio.com
